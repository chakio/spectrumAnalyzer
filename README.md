spectrumAnalyzer
====
* LEDアレイでスペクトラムアナライザー
![result](https://github.com/chakio/spectrumAnalyzer/blob/master/media/spectrumAnalyzer.gif) 
## Description
音響機器によく搭載sれていたスペクトラムアナライザーをarduinoとLEDバーで作成しました。
音の可視化方法のひとつとして作ってみたかったものです。
## Environment
* マイコン:arduino micro
* LED:WS2812B
